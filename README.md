# Простая Соц Сеть

Очень простая социальная сеть в стиле Twitter для небольшой группы друзей.

## Что это?

Это минималистичная социальная сеть где можно:
- Писать посты (только текст)
- Видеть посты других
- Удалять свои посты

## Как запустить?

### Шаг 1: Установить Node.js

Если у тебя еще нет Node.js, скачай его с сайта: https://nodejs.org/
Выбери версию LTS (рекомендуемая).

### Шаг 2: Установить зависимости

Открой терминал в папке проекта и выполни:

```
npm install
```

Эта команда скачает нужные библиотеки (Express - это библиотека для создания сервера).

### Шаг 3: Запустить сервер

Выполни команду:

```
npm start
```

Ты увидишь сообщение: "Сервер работает на http://localhost:3000"

### Шаг 4: Открыть в браузере

Открой браузер и перейди по адресу: http://localhost:3000

Готово! Теперь можешь писать посты.

## Как поделиться с друзьями?

### Вариант 1: Локальная сеть (если вы в одной WiFi сети)

1. Узнай свой IP адрес:
   - На Mac: открой "Системные настройки" → "Сеть" → посмотри IP адрес
   - Или в терминале выполни: `ifconfig | grep "inet "`
   
2. Друзья должны открыть в браузере: `http://ТВОЙ_IP:3000`
   Например: `http://192.168.1.100:3000`

### Вариант 2: Хостинг в интернете

Можешь задеплоить на бесплатные сервисы:
- **Glitch.com** - очень просто, просто загрузи файлы
- **Render.com** - тоже бесплатно
- **Railway.app** - еще один вариант

## Структура файлов

- `server.js` - сервер (обрабатывает запросы)
- `public/index.html` - страница которую видит пользователь
- `data.json` - файл где хранятся все посты
- `package.json` - список зависимостей

## Как это работает?

1. **Сервер (server.js)** - это программа которая работает на твоем компьютере и обрабатывает все запросы
2. **HTML страница** - это то что видит пользователь в браузере
3. **data.json** - это файл где хранятся все посты (как база данных, только очень простая)

Когда кто-то публикует пост:
- Браузер отправляет запрос на сервер
- Сервер сохраняет пост в data.json
- Сервер отправляет ответ обратно
- Страница обновляется и показывает новый пост

## Остановить сервер

Нажми `Ctrl + C` в терминале где запущен сервер.
